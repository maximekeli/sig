# Guide utilisateur — SIG Sols Togo (extrait)

## Connexion
1. Ouvrir http://localhost:8080
2. Saisir identifiant agent (`agent1` / `agent123`)
3. Cliquer **Connexion**

## Carte
- **Filtres** : pH, type de sol — **Appliquer**
- **Couches NASA** : cocher MODIS, SMAP, etc.
- **Couleur des points** : rouge (acide pH&lt;5,5), jaune (neutre), vert (basique)
- **Prédire fertilité** : saisir pH/humidité/type puis bouton

## Tableau de bord
KPIs : nombre de points, pH moyen, NDVI, zones dégradées, F1 modèle IA, corrélation SMAP.

## Quiz
1. Choisir niveau (facile / moyen / difficile)
2. **Démarrer** — 20 s par question
3. **Terminer** pour badges et score

## Badges
- Apprenti : 5 questions
- Connaisseur : 100 points cumulés
- Expert des sols : 300 pts + 3 sessions difficiles
- Champion NASA : toutes questions NASA correctes
