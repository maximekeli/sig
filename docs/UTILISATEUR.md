# Guide utilisateur — SIG Sols Togo

## Accès

- Application : http://localhost:8081/frontend/index.html
- Comptes démo : `admin` / `admin123`, `agent1` / `agent123`, `public1` / `public123`

## Fonctionnalités principales

### Carte
- Filtres pH, type de sol, validation
- Couches NASA, carte de chaleur, proximité
- Série NDVI par point (ID) dans la barre latérale
- Alertes sécheresse à proximité (après activation GPS)

### Mon espace
- Statistiques personnelles : points, vidéos, quiz, abonnés

### Vidéos & Shorts
- Catégories et hashtags
- Stories 24 h (agents connectés)
- Shorts en défilement vertical (molette)

### Communauté
- Fil d’abonnements, favoris, messages privés, profils publics

### Quiz
- Mode examen (120 s par question)
- Parcours pédagogique et défi hebdomadaire
- Certificat PDF avec code de vérification

### Notifications
- Cloche en en-tête (badge non lus)

### Recherche
- Barre globale : points, fiches, vidéos, utilisateurs

## Administration

- Journal modération unifié
- Import CSV utilisateurs
- Rapport ministère HTML
- Exports CSV activité / utilisateurs
